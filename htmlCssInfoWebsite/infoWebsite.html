Where to start?

You should start with finding a few youtube videos or websites to help you learn. Some good youtubers are SuperSimpleDev and Programming with Mosh. A few websites that you can learn from are freecodecamp and codecademy.

Which first?

Instead of learning one first, or one at a time. You should learn both at the same time as they go hand in hand!

Some of the best resources I've found are stack overflow and w3schools.

Basic HTML

Basic CSS

To target an tag in CSS you need to type the name of the tag such a p and then put {} after it. Then you can type in your designs such as color:blue to make the text color blue
